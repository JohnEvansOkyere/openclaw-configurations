# YouTube AI Channels to Monitor

These are the top AI/automation YouTube channels to scan daily for content ideas.
The agent browses these channels every evening and logs new videos.

## Channels

1. **Matt Wolfe** — https://www.youtube.com/@mreflow
   - Focus: AI tools, news, weekly roundups
   - Why: Huge audience, covers latest AI tools and trends

2. **Liam Ottley** — https://www.youtube.com/@LiamOttley
   - Focus: AI automation, n8n, agency building
   - Why: Closest to what John builds — AI systems for businesses

3. **Nicholas Renotte** — https://www.youtube.com/@NicholasRenotte
   - Focus: ML, deep learning, Python projects
   - Why: Strong technical content, good project ideas

4. **The AI Advantage** — https://www.youtube.com/@aiadvantage
   - Focus: Practical AI tools, workflows, productivity
   - Why: Practical, actionable content that performs well

5. **Fireship** — https://www.youtube.com/@Fireship
   - Focus: Tech news, dev tools, AI updates (fast-format)
   - Why: Master of hooks and short punchy content — great style reference

## Instructions for Agent
- Visit each channel page
- Find videos uploaded in the last 24-48 hours
- Log title, URL, view count, topic, and a 2-sentence summary
- Save all findings to: ideas/logs/youtube-video-log.md
