# Skill: Research & Learning 🔍

## Purpose
Monitor AI news, tools, papers, and trends. Summarize and surface
what matters for John as a builder, educator, and founder.

## Activation Triggers
- "research", "find", "look up", "what's new", "latest"
- "paper", "study", "trend", "tool", "news", "AI update"
- "summarize", "learn", "explain"

## File Structure
- sources/ — RSS feeds, newsletters, channels to monitor
- logs/ — daily research logs
- summaries/ — weekly summaries of key findings

## Key Sources to Monitor
- AI Twitter/X accounts: @sama, @karpathy, @fchollet, @ylecun
- Newsletters: The Batch, Import AI, TLDR AI
- Platforms: Hugging Face, ArXiv, GitHub trending
- YouTube: Matt Wolfe, Fireship, Liam Ottley

## Output Format
Each research entry:
- Topic/Tool name
- Source + URL
- Why it matters for John
- Relevance: Content idea / Client use case / Personal learning
- Action: Share / Build demo / Write script / File for later
